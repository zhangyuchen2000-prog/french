# Cinq

A small installable web app for learning five French words every day.

## Run locally
Open `index.html` directly, or serve the folder with a local web server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Publish with GitHub Pages
Upload all files to a GitHub repository, then enable **Settings → Pages → Deploy from a branch**.

Progress is stored in the browser's localStorage on each device.
