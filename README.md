# Cinq — Édito A1 complete core lexicon edition

This update keeps the latest Cinq interface and replaces the small demo list with a structured learning database based on all ten units of **Édito A1, 2e édition (2022)**.

## Imported

- **768 learning cards**
- **10/10 units covered**
- French term or communicative expression
- English meaning
- Chinese meaning
- unit and source label
- short practice example
- browser pronunciation
- random new groups of five
- unseen-first cycling within the selected unit
- unit filter in Settings
- daily progress, quiz and streak functions retained

## Scope

This app covers the textbook's **target vocabulary, practical expressions and main communicative language** across all ten units. It does not reproduce every word from every reading passage, instruction, transcript or copyright page. The example sentences in the app are newly written for practice rather than copied from the textbook.
